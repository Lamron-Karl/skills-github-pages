# 👋 Welcome

这里是我的个人知识库和在线文档中心，所有内容均由 **Markdown** 写成，通过 [Docsify](https://docsify.js.org) 生成，并部署在 [GitHub Pages](https://pages.github.com) 上。

## ✨ 站点特点

- **零门槛编写** – 只需掌握 Markdown，无需懂前端代码就能持续更新内容。
- **即时渲染** – 页面无需提前构建，保存文档后浏览器刷新即可看到最新效果。
- **清晰的侧边栏导航** – 左侧目录根据 `_sidebar.md` 自动生成，支持多级分类。
- **全文搜索** – 顶栏内置搜索功能，输入关键词即可在所有文档中快速定位。
- **响应式布局** – 手机、平板、电脑上都能获得不错的阅读体验。
- **完全免费托管** – 依托 GitHub Pages，零成本、免维护。

## 🧭 如何使用本站？

1. 查看左侧 **侧边栏**，点击感兴趣的文章标题即可阅读。
2. 在顶部 **搜索框** 中输入关键字，查找你需要的内容。
3. 想返回主页时，随时点击左上角的站点标题。
